# DriveDiary
## Волков Павел
Приложение для обслуживания автомобиля. В будущем здесь можно будет отслеживать состояние автомобиля. Следить за заменой масла в двигателе и коробке, относительным состоянием тормозных колодок с учётом стиля езды. Так же будет возможность оставлять заметки о поездке. Какие эмоции были получены, как себя вёл автомобиль, замечания.
